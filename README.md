A **text-based adventure game** in Python where your mission is to find the treasure.
This project is beginner-friendly and teaches **Python input, conditional statements, and game logic**.

---

## Features

* Interactive story with **multiple choices**
* Conditional logic with `if`, `elif`, `else`
* Input from the user to navigate the game
* Fun ASCII art for a treasure map

---

## How to Play

1. Run the program:

```bash
python treasure_island.py
```

2. Follow the prompts:

   * Choose directions at crossroads (`left` or `right`)
   * Decide whether to `wait` for a boat or `swim` across the lake
   * Pick the correct door (`red`, `yellow`, or `blue`) to find the treasure

3. The game ends if you make the wrong choice, or you win if you find the treasure.

---

